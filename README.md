Experimenting General Purpose GPU.

Specifically, we find the min and the max amongst a (sub-)array of floats.

```console
npm install
npm run sandbox
```

Example output:

```console
GPU: { min: 0.020119018852710724, max: 999.996337890625 } computed in 11.805 ms
CPU: { min: 0.020119018852710724, max: 999.996337890625 } computed in 18.534 ms
```
